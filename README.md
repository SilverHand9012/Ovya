# Ovya 🌸

Offline-first PCOS health companion for women.

## 🚀 Features

- Offline symptom tracking
- AI-powered insights (Gemini)
- Automatic sync to cloud (Firestore)
- Multilingual support
- Mental health tracking

## 🧠 Innovation

Works fully offline and syncs automatically when internet is restored.

## 🛠 Tech Stack

- Flutter
- Isar (local DB)
- Firebase Firestore
- Gemini AI

## 🎯 SDG Goal

Aligned with SDG 3: Good Health and Well-being
